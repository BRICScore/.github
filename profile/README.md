# BRICScore - a student graduation project organisation

Welcome to our student organization project. This repository contains materials, resources, and code developed as part of our academic work and collaborative activities.

## About

BRICS - Breathing Rhythm Identification and Classification System - project was created for research purposes and focuses on data processing and neural network development for classification and identification tasks. As part of the project we operate a specialized measuring appliance, process the raw data collected by it and finally use the data for ML purposes. In general the project serves as a platform for experimentation, analysis, and practical implementation of machine learning techniques within an academic environment.
